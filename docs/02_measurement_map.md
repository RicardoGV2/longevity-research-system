# Measurement Map

This file defines the universe of measurements that may be relevant to longevity and health functionality. It does not mean all measurements should be done immediately.

## Principle

First define what information would be scientifically useful. Later classify each measurement by:

- Priority.
- Feasibility.
- Cost.
- Invasiveness.
- Frequency.
- Risk of misinterpretation.
- Actionability.
